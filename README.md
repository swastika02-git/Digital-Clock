# ⏰ Digital Clock

A simple digital clock built using **HTML, CSS, and JavaScript** that displays the current time in **12-hour format** with an **AM/PM indicator**. The clock updates every second using JavaScript's `Date` object and `setInterval()`.

## 🚀 Features

- Displays the current time in real time
- 12-hour format with AM/PM
- Leading zeros for single-digit hours, minutes, and seconds
- Automatically updates every second
- Clean and responsive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. The clock will automatically display and update the current time every second.
